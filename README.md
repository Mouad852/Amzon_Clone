# 🛒 Amazon Clone

Ce projet est une réplique simplifiée de l’interface du site Amazon, développée à des fins d'apprentissage et de pratique front-end.

---

## ✨ Fonctionnalités

- Page d'accueil similaire à Amazon
- Liste de produits
- Panier d'achat
- Design responsive 

---

## 🧰 Technologies utilisées

- HTML
- CSS
- JavaScript
